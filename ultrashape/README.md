# ultrashape

## Purpose / scope
Ultrashape refinement utilities for asset processing.

## Primary entrypoints
- `ultrashape_refiner.py` main refiner.
- `__init__.py` package export.

## Required inputs / outputs
- **Inputs:** input assets or intermediate data used by the refiner.
- **Outputs:** refined ultrashape assets or metadata.

## Key environment variables
- Variables defining input/output locations and any external service configuration.

## How to run locally
- Run the refiner: `python ultrashape_refiner.py` (set required env vars).
