# fixtures

## Purpose / scope
Fixtures and generators for mock data and contracts used by tests and local development.

## Primary entrypoints
- `generate_mock_geniesim_local.py`
- `generate_mock_stage1.py`
- `contracts/` fixture contracts

## Required inputs / outputs
- **Inputs:** configuration for mock data generation.
- **Outputs:** generated mock assets and contract data used in tests.

## Key environment variables
- Any environment variables referenced by the generator scripts to locate output directories.

## How to run locally
- Run generators directly, for example: `python generate_mock_geniesim_local.py`.
